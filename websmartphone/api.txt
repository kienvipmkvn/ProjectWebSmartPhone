BRAND :

Lấy danh sách brand:
	method : GET
    url:localhost/websmartphone/api/brand/read.php    // lấy tất cả danh sách
	url:localhost/websmartphone/api/brand/show.php?id=1   // lấy theo id
	
thêm brand:
	[POST] localhost/websmartphone/api/brand/add.php
	post fields body : json :
		{
			"Name":"Name",
			"Detail":"Detail"
		}
sửa brand:
	[PUT] localhost/websmartphone/api/brand/update.php
	post fields body : json :
		{
            "ID":"ID",
			"Name":"newName",
			"Detail":"newDetail"
		}
xóa brand:
	[DELETE] localhost/websmartphone/api/brand/delete.php
    post fields body : json :
        {
            "ID":"ID"
        }	


PRODUCT

Lấy danh sách product:
	method : GET
    url:localhost/websmartphone/api/product/read.php?id=page     // lấy danh sách theo từng trang với 10 phần tử. VD page=1 thì lấy 10 phần tử đầu tiên của database
	url:localhost/websmartphone/api/product/show.php?id=1     // lấy theo id
	url:localhost/websmartphone/api/product/search.php?name=name  // tìm kiếm product dựa theo từ khóa, ví dụ: từ khóa iphon ->> tìm ra các product có tên iphone 5s, 6s v.v..
	url:localhost/websmartphone/api/product/sbrand.php?name=name // lấy ra các product dựa theo từ khóa tên của brand. ví dụ từ khóa black ->> blackberry.
	
thêm product:
	[POST] localhost/websmartphone/api/product/add.php
	post fields body : json :
		{
			"Name":"Name",
			"Detail":"Detail",
            "Price" : Price,
            "ImageLink" : "ImageLink",
            "BrandID" : "BrandID"
		}
sửa product:
	[PUT] localhost/websmartphone/api/product/update.php
	post fields body : json :
		{
            "ID":"ID",
			"Name":"newName",
			"Detail":"newDetail",
            "Price" : newPrice,
            "ImageLink" : "newImageLink",
            "BrandID" : "newBrandID"
		}
xóa product:
	[DELETE] localhost/websmartphone/api/product/delete.php
    post fields body : json :
        {
            "ID":"ID"
        }
